# 🏆 Fortnite Live Tournament Tracker

Aplikacja w Streamlit służąca do zaawansowanej analityki turniejów Fortnite (Duos) bezpośrednio z danych kopiowanych z Fortnite Tracker po każdej grze. 
Aplikacja automatycznie rozbija skumulowane statystyki i wylicza dokładny przebieg poszczególnych meczów.

## 🚀 Jak uruchomić projekt lokalnie?

1. Sklonuj to repozytorium lub pobierz pliki na dysk.
2. Upewnij się, że masz zainstalowanego **Pythona 3.8+**.
3. Zainstaluj wymagane biblioteki, wpisując w terminalu/konsoli:
   ```bash
   pip install -r requirements.txt
